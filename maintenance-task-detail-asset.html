Start On
  • This is the first date maintenance is due.
  • It’s used when the task happens only once (e.g., initial setup inspection).

Example:
Inspection starts on Jan 1, 2025 → next due = Jan 1, 2025

Repeat Every
  • This is used when you want maintenance to happen repeatedly.
  • Based on the last inspection date or "Start On" date if it’s the first one.

Example:
Repeat every 90 days → Jan 1, Apr 1, Jul 1...

First-Time On + Repeat Every
  • Use this if you want the first maintenance to start on a specific date, and then repeat regularly.
  • It will calculate all next dates based on that first date + repeat.

Example:
First Time On: Feb 15, 2025
Repeat Every: 6 months → next: Aug 15, 2025, then Feb 15, 2026...
